Часто задаваемые вопросы о сводной информации о лицензиях

Список лицензий показывает актуальное количество корпоративных лицензий в каждом семействе продуктов Майкрософт с разбивкой по версиям для всех действующих и неактивных идентификаторов лицензий в вашем профиле. При создании списка лицензий учитывается много факторов, среди которых: правила лицензирования; даты выпуска продуктов; приобретенные полные лицензии, лицензии на обновление и обслуживание. Примечание. Полные лицензии, приобретенные с определенным идентификатором лицензии, приводятся в списке лицензий, только если пользователь видит этот идентификатор лицензии в сводной таблице. (Иными словами, только если пользователь, вошедший в VLSC, имеет разрешение на просмотр элементов, связанных с этим идентификатором лицензии.) При просмотре отдельного идентификатора лицензии в сводной таблице отображаются только покупки по данному соглашению. (Значения в столбце "Неполные лицензии" могут быть неверными.) Чтобы открыть и просмотреть список лицензий на веб-сайте VLSC, щелкните Лицензии, а затем щелкните Список лицензий.

Возможные причины: у корпорации Майкрософт нет информации о полных лицензиях, приобретенных клиентом у поставщиков техники (OEM) или в виде коробочных версий; в случае неиспользованного покрытия лицензий на обновление. Например, лицензии Maintenance, Upgrade Advantage или Software Assurance будут указаны в качестве неполных, так как действительные основные лицензии не были обнаружены.

Для каждой основной лицензии, для которой имеется подтверждение права на лицензию, число в столбце "Неполные лицензии" может считаться фактическим количеством.

В категории лицензий лицензии группируются по типу продукта. В программах корпоративного лицензирования Майкрософт продукты подразделяются на три категории: приложения (например, выпуски Microsoft Office); системы (например, версии операционной системы Windows); серверы (например, Exchange Server, SQL Server и Windows Server).

Недавно приобретенные лицензии обычно появляются в списке лицензий в течение 24 часов с момента получения Майкрософт заказа от партнера. Возможна задержка от нескольких дней до нескольких недель в том случае, если партнер, у которого вы приобрели лицензию, задерживает перенаправление заказа на поставку в Майкрософт.

Время от времени в рамках программ корпоративного лицензирования Майкрософт проводятся рекламные акции. Участники таких акций, которые приобрели определенный продукт, получают право на использование другого продукта. Рекламные акции, как правило, зависят от действующего покрытия Software Assurance или Upgrade Advantage, распространяющегося на приобретенный продукт. Например. Клиент приобретает одну лицензию на Exchange Server 2016. В период проведения рекламной акции на эту транзакцию распространяется действующее покрытие Software Assurance или Upgrade Advantage. В рамках рекламной акции за приобретение лицензии на Exchange Server 2016 Майкрософт предлагает одну лицензию на Skype для бизнеса Server. Предоставление лицензии дает клиенту право использовать продукт, предлагаемый в рамках рекламной акции. В этом случае клиенту предоставляется право на использование Skype для бизнеса Server. Дополнительные сведения о предоставлении лицензий см. в документе Права на использование продуктов в рамках программ корпоративного лицензирования Майкрософтоткрывается в новом окне и в прайс-листах.

Сведения о предоставлениях лицензий находятся в разделах Relationship Summary (Сводная таблица), License Summary (Список лицензий) и Product Downloads (Загрузка продуктов). В разделах Relationship Summary (Сводная таблица) и License Summary (Список лицензий) продукты, для которых имеются предоставления лицензий, отмечены звездочкой (*), обозначающей, что продукт не был приобретен напрямую.

Страница Резервирование лицензий — это функция резервирования лицензий через Интернет на веб-сайте VLSC. На этой странице можно разместить заказ на резервирование лицензий, чтобы перенести локальные ресурсы в облако без оформления заказа на поставку. Резервирование лицензий предусматривается только для служб Online Services, если в рамках соглашения предоставляется такое право. Ниже перечислены еще некоторые особенности заказов на резервирование. Заказ на резервирование является финансовым обязательством, которое реализуется в рамках процессов размещения заказов True Up или годовых заказов. При этом учитываются дата использования резервирования и зарезервированное количество. Заказы на резервирование можно размещать на текущую дату или для использования в будущем; датой использования может быть любая дата в течение следующих шести месяцев. Заказ на резервирование можно отменить в течение 72 часов после даты использования (включая выходные и праздничные дни). Счета за заказы на резервирование, размещенные в течение 30 дней до годовщины, выставляются в следующем году.

Корпорация Майкрософт отказалась от процедуры резервирования QuickStart в мае 2016 года. Исходя из отзывов клиентов, корпорация Майкрософт оптимизировала процедуру резервирования. Теперь используется одна упрощенная процедура. Соответствующие клиенты теперь могут резервировать любые доступные службы и выполнять их согласование в составе заказов True Up или годовых заказов.

В списке соглашений Enterprise Agreement для резервирования лицензий выберите действующее соглашение, чтобы продолжить резервирование через Интернет, а затем нажмите кнопку Continue (Продолжить). Выберите дату использования для резервирования через Интернет, а затем нажмите кнопку Continue (Продолжить). Выберите нужную службу, а затем действие, которое требуется выполнить для этой службы: Add License (Добавить лицензию) или Step-up License (Повысить уровень лицензии).
