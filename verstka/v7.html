Часто задаваемые вопросы по Software Assurance

Программа Software Assurance для корпоративного лицензирования помогает повысить производительность во всей организации путем получения максимальной отдачи от использования ПО Майкрософт. Преимущества, включая круглосуточную поддержку, услуги по планированию развертывания, обучение конечных пользователей и технических специалистов, самые последние версии ПО и уникальные технологии объединены в одну рентабельную программу. Использование этих преимуществ может помочь вашей организации улучшить производительность и помочь ИТ-специалистам эффективно развертывать и управлять технологиями Майкрософт.

Эти преимущества могут быть различными в разных программах корпоративного лицензирования (например, соглашении Enterprise или Open Value Agreement) и зависеть от числа соответствующих требованиям лицензий, зарегистрированных в программе Software Assurance. Эти преимущества можно просмотреть на странице программы Software Assurance в центре Volume Licensing Service Center (VLSC). Для управления преимуществами Software Assurance через VLSC необходим Internet Explorer 5 или более поздней версии. Вы можете использовать ресурсы и средства с сайта Software Assurance, чтобы узнать больше о преимуществах Software Assurance, доступных вашей организации.

Изучите следующие доступные для загрузки руководства по программе Software Assurance на сайте VLSC. На этой странице отображены сводные данные о преимуществах Software Assurance, которые доступны для активации и управления в режиме онлайн с помощью идентификатора лицензии , к которому у вас есть доступ. Преимущества, не требующие активации или управления, не включены в сводные сведения о Software Assurance. Для каждого преимущества, включенного в сводные сведения, отображаются следующие значения: общее доступное количество; количество лицензий, доступных для использования; количество лицензий, требующих предварительной активации.

На этой странице представлен список преимуществ Software Assurance, доступных вам по конкретным идентификаторам лицензий. Для преимуществ также отображаются следующие атрибуты, связанные с идентификатором лицензий. Если вы соответствуете требованиям, то в VLSC для активации и управления доступны следующие преимущества Software Assurance: услуги по планированию‎; Microsoft Desktop Optimization Pack (MDOP); пакет многоязычного интерфейса для Office*; ваучеры на обучение; E-Learning; программа использования ПО на домашних компьютерах (HUP); круглосуточная поддержка в решении проблем. * Для набора Office 2016 пакет многоязычного интерфейса для Office является частью набора; скачивать его из VLSC не требуется.

Столбец состояния на странице "Управление преимуществами Software Assurance" отображает состояние преимущества, связанного с определенным идентификатором лицензии. На странице "Управление преимуществами Software Assurance" возможны следующие состояния: "Требует активации". Менеджер Software Assurance должен активировать преимущество, прежде чем его можно будет использовать. "Активировано". Менеджер Software Assurance активировал преимущество, что позволяет теперь его использовать. "Ожидание повторной активации". Преимущество SA было приостановлено Майкрософт. "Истекло". Покрытие программой Software Assurance истекло и дальнейшее действие не разрешено.

Зарегистрированные пользователи VLSC, которым для определенного идентификатора лицензии назначена роль менеджера Software Assurance, могут управлять преимуществами Software Assurance, связанными с этим идентификатором лицензии. Администратор для данного идентификатора лицензии также имеет право действовать в качестве менеджера Software Assurance для данного идентификатора лицензии. Роль менеджера Software Assurance назначается администратором указанного идентификатора лицензии. Кроме того, Майкрософт может автоматически назначить роль менеджера Software Assurance на основании той информации, которую ваша организация передала с помощью формы для дополнительной контактной информации во время установления идентификатора лицензии.

Менеджер Software Assurance может выполять задачи, связанные с управлением преимуществами SA, которыми ему было поручено управлять. У администратора больше возможностей, чем у менеджера Software Assurance. Роль администратора позволяет назначать разрешения пользователям VLSC, просматривать информацию о лицензии, загружать ПО, просматривать ключи продуктов, управлять подписками, управлять службами Online Services, а также управлять преимуществами Software Assurance для каждого идентификатора лицензии в их наборе разрешений. Дополнительные сведения о ролях и разрешениях пользователей, которые могут назначаться в VLSC см. в часто задаваемых вопросах о разрешениях.

Менеджер Software Assurance может активировать и управлять каждым преимуществом, специально назначенным администратором. В зависимости от особенности преимущества задачи управления могу включать в себя установление соответствующими доменами, отправку и прием физических носителей, загрузку подходящего программного обеспечения для преимуществ и назначение и (или) передачу ваучеров. Примечание. Для каждого идентификатора лицензии менеджер Software Assurance может быть ограничен в управлении только одним или большим количеством подходящих преимуществ, связанных с этим идентификатором лицензии.

На странице Мои разрешения отображаются все роли, которые назначены вам для каждого идентификатора лицензии. Если выбрано (отмечен флажок) ваше назначение в качестве менеджер Software Assurance, то вы можете управлять всеми подходящими преимуществами. Если флажок снят, вы можете управлять только некоторыми подходящими преимуществами. Наведите указатель мыши на поле, чтобы увидеть права, которые были вам предоставлены.
